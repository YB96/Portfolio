Exploratory Data Analysis on Brazilian E-Commerce Olist

This is a Brazilian ecommerce public dataset of orders made at Olist Store. The dataset has information of 100k orders from 2016 to 2018 made at multiple marketplaces in Brazil. Its features allows viewing an order from multiple dimensions: from order status, price, payment and freight performance to customer location, product attributes and finally reviews written by customers. It has also been included a geolocation dataset that relates Brazilian zip codes to Lat/Lng coordinates.
Olist is a Brazilian departmental store (marketplace) that operates in e-commerce segment, but is not an e-commerce itself. It operates as a SaaS (Software as a Service) technology company since 2015.

Automatic Sudoku Solver

Sudoku is a logic-based, number-placement puzzle. The objective is to fill a 9x9 grid with numbers so that each column, each row, and each of the nine 3x3 sub-grids that compose the grid contain all of the digits from 1 to 9.
Sudoku puzzles can range in difficulty from easy to extremely challenging. Some puzzles can be solved using simple logic, while others require advanced problem-solving techniques. The puzzle originated in Japan, but has gained worldwide popularity. It can be played on paper or using computer programs and mobile apps. Some variations of the game include using letters or symbols instead of numbers, and using a different size grid.
